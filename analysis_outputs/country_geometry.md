## Interpretation

At h2, country is not encoded as a single linear direction. One more learned affine map plus ReLU makes it linearly available at h3. That means the h2 representation is a piecewise-linear region: the following layer applies several half-space gates, and the final country logit combines those gates. The best description is therefore not a single direction, but a gated pocket or union of gated pockets in h2 activation space.

The `country_h3_unit_contributions.csv` file shows which ReLU gates carve out this region. The `country_top_h3_units_recover_label.csv` file shows whether a few gates are enough or whether the country region is distributed across many gates.
